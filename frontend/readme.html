  2. Server crash glitch (no more ragequits)

Chat Features

  • Markdown Support: Format your messages with **bold**, ~~italics~~, __underline__, --strikethrough--, ^^big text^^, $r$rainbow text$r$, ||spoilers||, and ``code``
  • Context Menu: Right-click on any Bonzi for quick actions
  • Quote System: Quote the last message with the context menu
  • Direct Messages: Send private messages to other users
  • Media Support: Share images and videos from catbox.moe

Commands

Here are all the commands you can use. The square brackets indicate parameters, please don't include them!

Basic Commands

  • /name [new name] - Change your name (supports markdown!)
  • /color [color] - Change your Bonzi's color (purple, blue, red, green, pink, black, brown)
  • /pitch [number] - Change TTS pitch (0-400)
  • /speed [number] - Change TTS speed (0-400)

Fun Commands

  • /joke - Bonzi tells a joke (warning: they're terrible)
  • /fact - Bonzi shares a "fun" fact
  • /backflip [swag] - Bonzi does a backflip (add "swag" for extra coolness)
  • /asshole [username] - Call someone an asshole (classy)
  • /owo [username] - Notice someone's bulge (owo what's this?)
  • /triggered - The classic triggered copypasta
  • /linux - Richard Stallman would be proud
  • /quote - Quote the last message sent

Media Commands

  • /image [catbox URL] - Display an image in chat (catbox.moe only)
  • /video [catbox URL] - Play a video in chat (catbox.moe only)
  • /youtube [video ID] - Play a YouTube video (just paste the URL, we'll figure it out)

Admin Commands (Level 2)

  • /godmode [password] - Get admin privileges (check config.json for password)
  • /pope - Turn into the Pope (requires godmode)
  • /restart - Restart the server (assumes auto-restart is enabled)
  • /update - Reload config files without restart
  • /sanitize - Toggle text filtering on/off

Context Menu Actions

Right-click on any Person to access:
  • Cancel - Stop whatever that guy is doing
  • Mute/Unmute - Someone spamming the /linux copypasta? Mute em.
  • Call an Asshole - Quick way to be rude
  • Notice Bulge - The classic owo
  • Quote - Quote the last message
  • DM - Send a direct message

Markdown Guide

Make your messages fancy with these formatting options:
  • **bold text** - Makes text bold
  • ~~italic text~~ - Makes text italic
  • __underlined text__ - Underlines text
  • --strikethrough text-- - Crosses out text
  • ^^big text^^ - Makes text bigger
  • $r$rainbow text$r$ - Makes text rainbow colored
  • ||spoiler text|| - Hides text until clicked
  • ``code text`` - Formats as code

Setup

  1. Make sure you have Node.js installed (because we're not savages)
  2. Run npm install to get the dependencies
  3. Edit config/config.json to customize settings
  4. Add colors to config/colors.txt (one per line)
  5. Add banned words to config/blacklist.txt (one per line)
  6. Run node index.js to start the server
  7. Profit!

Configuration

Edit config/config.json to customize:
  • port - Server port (default: 80)
  • godword - Admin password for /godmode
  • altlimit - Maximum connections per IP
  • slowmode - Cooldown between messages (ms)
  • namelimit - Maximum name length
  • charlimit - Maximum message length
